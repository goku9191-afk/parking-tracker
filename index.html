📋 설정 방법 (약 5분)

  1. console.firebase.google.com 접속 후 구글 로그인
  2. 프로젝트 만들기 → 이름 입력 → 프로젝트 만들기
  3. 빌드 → Realtime Database → 데이터베이스 만들기 → 테스트 모드
  4. 프로젝트 설정 → 웹 앱 추가 (</>) → firebaseConfig 코드 복사
  5. 이 HTML 파일 메모장으로 열어 YOUR_로 시작하는 값들을 교체 후 저장
