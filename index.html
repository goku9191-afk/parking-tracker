📋 설정 방법 (약 5분)

  1. console.firebase.google.com 접속 후 구글 로그인
  2. 프로젝트 만들기 → 이름 입력 → 프로젝트 만들기
  3. 왼쪽 메뉴 빌드 → Realtime Database → 데이터베이스 만들기 → 위치 아무거나 → 테스트 모드로 시작
  4. 프로젝트 홈 → 톱니바퀴 → 프로젝트 설정 → 내 앱 → 웹 앱 추가 (</>) → 앱 닉네임 입력 → 앱 등록
  5. firebaseConfig 코드 복사
  6. 이 HTML 파일을 메모장으로 열어 YOUR_로 시작하는 값들을 교체 후 저장
  7. 친구와 공유할 때: 이 파일을 index.html로 이름 바꾸기 → github.com에서 새 저장소 만들고 파일 업로드 → vercel.com에서 그 저장소 import → Deploy → 생성된 링크 공유
